This is a simple and attractive weather application which show the basic details of weather across the world.
This page shows the deatils like
                    Temperature
                    Time
                    Date,Day and year
                    Humidity
                    Cloud Percentage
                    Wind Speed
The web application also includes Search bar to search the cities.
